PM2.5 농도의 전문 측정
KJ1은 일반 가정에서도 사용할 수 있도록 특별히 재설계된 전문 PM2.5 입자 모니터입니다.
이 내구성 강한 기기는 빠르고 정확하게 PM2.5 농도를 측정하여 전문가 수준의 정확도를 제공합니다.
KJ1은 일반 가정에서도 사용할 수 있도록 특별히 재설계된 전문 PM2.5 입자 모니터입니다. 이 내구성 강한 기기는 빠르고 정확하게 PM2.5 농도를 측정하여 전문가 수준의 정확도를 제공합니다.
간단하고 편리한 사용
KJ1 측정기의 ON/OFF 키를 몇 초간 눌러 켜면, 기기가 자동으로 3분마다 PM1.0, PM2.5, PM10의 공기 농도를 측정합니다. 각 측정은 60초 동안 지속됩니다. PM 농도 수치와 함께 PM2.5의 오염 정도를 색상 코드로 표시하여, 일반인도 쉽게 공기 질을 이해할 수 있도록 합니다.
넓은 범위의 입자 측정
KJ1 입자 측정기는 PM1.0, PM2.5, PM10 입자의 농도를 0-9999μg/m³ 범위 내에서 빠르고 정확하게 측정할 수 있습니다. 측정 결과는 큰 화면에 쉽게 읽을 수 있도록 표시됩니다.
전문적인 데이터 계산
KJ1 측정기의 TEST 키를 길게 눌러 입자 제거 효율 측정 인터페이스로 들어갑니다. TEST 키를 눌러 초기 PM0.3-0.5/PM2.5 농도를 30초 동안 측정합니다. 다시 TEST 키를 눌러 공기 정화 후 PM 농도를 30초 동안 측정합니다. 기기는 자동으로 공기 청정기의 입자 제거 효율을 계산합니다.
