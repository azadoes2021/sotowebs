U8 UV lamp controller

U8은 새로운 지능형 UV 램프 컨트롤러로, 세심한 관찰을 통해 혁신과 개선을 이루었습니다. 고객에게 더 높은 수준의 서비스를 제공하기 위해 개발된 U8 지능형 UV 램프 컨트롤러는 지능형 운영 시스템과 레이더 센서를 갖추고 있으며, 작동 시 자동으로 소독 시간을 기록하고, 원클릭 내보내기 기능을 제공합니다.

U8 UV lamp controller

U8은 새로운 지능형 UV 램프 컨트롤러로, 세심한 관찰을 통해 혁신과 개선을 이루었습니다. 고객에게 더 높은 수준의 서비스를 제공하기 위해 개발된 U8 지능형 UV 램프 컨트롤러는 지능형 운영 시스템과 레이더 센서를 갖추고 있으며, 작동 시 자동으로 소독 시간을 기록하고, 원클릭 내보내기 기능을 제공합니다.

  • 사양
레이더 감지 기능
레이더 감지 기능이 있어 사람이 접근하면 자외선이 자동으로 꺼지고 작동이 중지됩니다. 사람이 떠난 후에는 설정된 절차에 따라 자동으로 소독을 재개하여, 다시 켜는 번거로움을 없앱니다.
작동 시간 설정
UV 램프의 작동 시간을 쉽게 설정할 수 있으며, 작업이 완료되면 자동으로 램프가 꺼집니다. 이를 통해 UV 램프를 끄는 것을 잊어서 발생할 수 있는 UV 노출 사고를 방지할 수 있습니다.
비밀번호로 설정
UV 램프를 켜려면 올바른 비밀번호를 입력해야 하며, 이를 통해 실수로 UV 램프를 켜는 것을 방지하고 우발적인 자외선 노출 사고를 예방할 수 있습니다.
살균소독 기록 조회 및 내보내기
U8은 데이터 처리 기능을 갖추고 있습니다. 각 작업마다 자동으로 소독 기록이 생성되며, 언제든지 이 기록을 조회할 수 있습니다. 또한, 소독 기록을 빠르게 내보낼 수 있어 수동으로 기록하는 번거로움을 덜어줍니다
